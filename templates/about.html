About Us

Our Project

This project is a final year graduation project focused on email spam detection using machine learning. Our system helps users identify potentially harmful or unwanted emails by analyzing the text content and determining whether it's likely to be spam or legitimate (ham).

We've built this application using modern web technologies and machine learning techniques to provide an accurate, fast, and user-friendly spam detection service that can be integrated with various email systems.

Technology Stack

Our project utilizes the following technologies:

  • Backend: Flask, Flask-RESTx, SQLAlchemy, JWT
  • Frontend: HTML, CSS, JavaScript, Bootstrap, Chart.js
  • Machine Learning: Scikit-learn, NLTK, TF-IDF
  • Database: SQLite (development), PostgreSQL (production)
  • Deployment: Heroku, Docker
